# 逃离鸭科夫Mod
- MiniMap：小地图
- BetterModUpload：更好的Mod上传，支持忽略某些特定文件
- TestMod：测试用，没什么功能
- ZoinkModdingLibrary：通用库，非Mod

## 开发环境
- 工具：Visual Studio
- 框架：.Net Standard 2.1

## 准备工作
- 下载本仓库并解压至任意文件夹
- 在解决方案文件 *ZoinkMods.sln* 同级目录下新建一个 *SteamLibrary.txt* 文件。
- 将你的Steam库目录写入上述文件，如：```D:\SteamLibrary\steamapps``` （以 steamapps 结尾）

## 开发
- 修改或添加你自己喜欢的Mod功能
